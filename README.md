# Auto Post Blog

AIを活用した自動ブログ記事生成・投稿システム

## プロジェクトの目標

**複数の特化ブログを作成し、Google AdSenseによる広告収入で収益化を実現する**

- ジャンル特化型ブログを複数運営（グルメ、ガジェット、旅行、買い物など）
- AIによる記事生成の自動化で、質の高いコンテンツを効率的に量産
- **主要収益源**: Google AdSense（広告収入）
- **副次収益**: Amazonアフィリエイト、ASP案件
- real-voice.comドメイン配下で統一ブランドを構築

### 🎯 最重要: SEO最適化

**全ての検討・実装において、SEO最適化を最優先で考慮する**

AdSense収益はトラフィック量に直結するため、SEO対策が成功の鍵となります：

1. **記事生成時のSEO対策**
   - AIプロンプトでSEOを意識したタイトル・見出しを生成
   - 検索意図に合致した網羅的なコンテンツ（1000-2000文字）
   - 適切なキーワード密度と共起語の使用
   - メタディスクリプション、OGP画像の自動生成

2. **サイト構造のSEO対策**
   - セマンティックHTML（適切なh1-h6階層）
   - 内部リンク構造の最適化
   - パンくずリスト
   - サイトマップ自動生成
   - モバイルファーストデザイン

3. **技術的SEO対策**
   - ページ速度の最適化（Cloudflare Pages活用）
   - 画像の最適化（WebP変換、遅延読み込み）
   - 構造化データ（JSON-LD）の実装
   - Core Web Vitalsの改善
   - SSL/HTTPS対応（Cloudflare標準）

4. **コンテンツ戦略**
   - ロングテールキーワードを狙った記事作成
   - 定期的な記事更新による鮮度維持
   - ユーザーの検索意図に応える質の高い情報提供
   - E-E-A-T（経験・専門性・権威性・信頼性）の向上

## 主な機能

### 📝 AIインタビュー形式の記事生成

- 初回フォーム入力（ジャンル、日付、タイトル案、本文、画像、費用、星評価）
- AIが追加質問を動的に生成（text/select/image形式）
- ユーザーの回答を基に、より詳細で魅力的な記事を自動生成

### ⭐ 星評価システム

- 1-5星の評価を各記事に追加
- 記事ページと一覧ページで視覚的に表示
- 評価でフィルタリング可能

### 🔍 高度な検索・フィルタリング

- テキスト検索（タイトル・カテゴリ）
- 年月フィルター
- 評価フィルター（5星、4星以上、3星以上）
- リアルタイムで絞り込み

### 📱 Sveltia CMS統合

- `/admin/`からブラウザで記事を編集・管理
- GitHub OAuth認証
- Markdownエディタ、画像アップロード対応
- AI生成記事の手動修正が可能

## 技術スタック

- **フロントエンド**: Hono + JSX, htmx
- **バックエンド**: Cloudflare Workers
- **AI**: Workers AI (Llama 4 Scout)
- **ストレージ**: KV Namespace (セッション管理)
- **ブログ**: Astro + Cloudflare Pages
- **CMS**: Sveltia CMS
- **認証**: GitHub OAuth

## セットアップ

### 1. 投稿システムのデプロイ

```bash
cd apps/worker
npm install
npm run deploy
```

### 2. 環境変数の設定

Cloudflare Dashboardで以下を設定：
- `GITHUB_TOKEN`: GitHubのPersonal Access Token（repo権限）
- `SESSIONS`: KV Namespace

### 3. ブログのデプロイ

各ブログリポジトリをCloudflare Pagesに接続：
- `blog-food` → food.real-voice.com
- `blog-gadget` → gadget.real-voice.com
- `blog-travel` → travel.real-voice.com
- `blog-shopping` → shopping.real-voice.com

## 収益化戦略

### 主要収益源: Google AdSense 💰

**SEO対策により検索流入を最大化し、広告収入を得る**

- 各ブログに自動広告を配置
- トラフィック増加 = 収益増加
- **成功の鍵はSEO最適化**によるオーガニック検索流入

### 副次収益

1. **Amazonアフィリエイト** - 商品レビュー記事に購入リンク
2. **A8.net等のASP** - ジャンル特化の案件を掲載
3. **独自商品・サービス** - 将来的に展開可能

### KPI（重要指標）

- **オーガニック検索流入数**: SEO効果の測定
- **ページビュー数**: AdSense収益に直結
- **直帰率・滞在時間**: コンテンツ品質の指標
- **検索順位**: ターゲットキーワードのランキング
- **AdSense収益**: 最終目標値
